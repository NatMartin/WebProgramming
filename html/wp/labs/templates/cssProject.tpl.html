CSS Project

Use CSS to add style to your index.

  1. Create a directory for all of your CSS files called css in the html directgory.
  2. Create a CSS file in the css directory called main.css.
  3. Change the page titles on both index files to have be surrounded by a light gray box (color #EEEEEE) surrounded by a border that is 3 pixels wide, a radius of 4 pixels, a dark gray border (color #333333). Pad to 10 pixels.
  4. Change the color of the text on the page titles to #333333.
  5. Surround the IndexItems with a border like the page title and set the background to the same color as the page titles.
  6. Remember to check your work into Git.
