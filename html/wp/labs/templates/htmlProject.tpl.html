HTML Project

Using HTML create an index for your web site.

  1. Create a file called index.html in /src/WebProgramming/server/html.
  2. Give the page specified by index.html the title: <name>'s web site.
  3. Create a centered h1 heading at the top of your page saying: <your name>'s web site. Give this item the name "WebSiteIndex" and the class "PageTitleText". Put the heading in a div with the class "PageTitle".
  4. Add an unordered list that has one item in it: an anchor that will load WPprojects/index.html when it is clicked. Give the anchor the value: Web Programming Projects. Give the ordered list the name "WebSiteIndexItems" and the class "IndexItems". Give the item the name "ProgramingProjectsIndexItem" and the class "IndexItem". Note the list's class (WebSiteIndexItems) is the plural of the items in the list (WebSiteIndexItem). The name of the items reflects their meaning.
  5. Create a directory called WPprojects in /src/WebProgramming/server/html.
  6. Create a new html file called index.html in /src/WebProgramming/server/html/WPprojects
  7. Create a centered H1 heading at the top of the second web page saying: <name>'s Projects.
  8. Add an ordered list that has one item in it: an anchor that will load WPprojects/index.html when it is clicked. Give the anchor the value: Web Programming Projects. Give the list and its items the same classes as the previous list. Given the list the name "ProgrammingProjects" and the item "P1_TOC".
  9. If you have not done so already, create a Git repository for WebProgramming and check everything in.
