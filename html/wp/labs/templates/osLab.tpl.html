Getting Ready

In this class, we will be using Unix-like operating systems because:

  • Probably most servers are hosted on Unix-like operating systems, it is hard to say for sure.
  • Windows version of server programming hides a lot of the details of how the server works; it is easier to understand how the client and server work together on Unix-like systems.

If you have a Macintosh, you are in luck, OS X is a Unix-like system. You will have to learn to user the terminal program, but you do not need to install a new OS. If you have a Windows system, you will need to install Linux. Fortunately, this is not particularly hard, and it only needs to be done once.

On the Macintosh

Getting ready to user a Unix-like system on the Mac is easy. All you need to do is find the Terminal program and pin it to your command bar.

  1. open your Applications folder
  2. open the Utilities folder in the Applications folder
  3. double click on the Terminal program
  4. A terminal program will open

On the PC

On the PC, things are more difficult. First, you are going to have to install Linux on your computer. After that, the computer will ask you which operating system you want to use when you log in. If you choose Linux, you will get a Linux GUI. You can then start a terminal window from the GUI. The following tutorial will guide you through downloading and installing Ubuntu Linux on your computer. There are many different versions of Linux; currently Ubuntu is the easiest to install and use.

Install Linux

Linux comprises many distributions. A distribution is a copy of the core of the operating system--the Linux Kernel--and the programs that run on top of it--almost always the GNU stack. In Linux, and almost all modern operating system, consist of a small core program that does the work that needs to be done without interference from other program such a switching from one program to another when multitasking. On top of the kernal, other programs do most of the work that we interact with. For example, the file system is a program that runs on top of the kernal. Distributions provide ways to install the operating system and support for it such as user groups. They usually fund themselves through consulting about the operating system.

Currently, as of 2014, Ubuntu is probably the easiest to install. That being said, it is not particularly easy because there are a many different software systems that have different requirements. If you have a older windows operating system (pre Windows 8) you are in luck; it is relatively easy to install. Get Ubuntu Linux

  1. Go to www.ubuntu.com.
  2. Click on "Download Ubuntu."
  3. Click on "Ubuntu Desktop"
  4. Choose the flavor from the drop down. You can find out what type of computer you have by going to the Control Panel > System and Security > System, or click on My PC (or this PC) and select properties. It will tell you if you have a 64 bit computer. In general it will be:
    • 64-bit if you have a relatively new computer.
    • 32-bit computer if you have an old one.
    • 64-bit Mac (AMD64) if you have a new Apple Computer, but if you have a new Apple Computer running OS X you already have unix running, so you don't need it.
  5. Click on Download

Back up your computer

To back up your computer you will need at least an 8 GB pen drive and probably a 16 GB pen drive. You can also back up to an external hard drive.

On Windows 8.1
  1. On the Charms Bar select search. (You get to the Charms Bar by moving the mouse to the upper left or right hand window corner or clicking Windows-C.
  2. Search for File History
  3. a Window will open shoing File History
  4. Turn on File History by clicking the Turn On button on the lower right hand side.
  5. It is probably a good idea to back up your operating system too.
Create a bootable memory stick

Before you start, make sure that there is nothing you need on the pen drive you are going to use. Everything on the pen drive will be erased.

  1. Go to www.pendrivelinux.com and download the UUI.
  2. When the .exe file loads, run it.
  3. Click I agree on the license agreement
  4. Choose Ubuntu as your operating system.
  5. Choose the file that contain the Ubuntu Operating System
  6. Select your USB drive and click Create
  7. A summary screen appears, cleck Yes.
Partition your Hard Drive to give Linux space
  1. Find the Charms Bar and Search for Partitions. Choose Create and format partitions when it appears.
  2. When the Disk Management screen appears, right click on the OS (C:) line and select Shrink volume
  3. It will scan the disk and give you a box that will tell you the amount of space on the disk, the amount by which you can shrink it and the size after the shrink. Choose a number less than or equal to the space to shrink in MB. I chose to shrink by the maximum amount.
  4. A lot of unallocation space will appear. Ubuntu will be installed there.

Install the Operating System

You will need to know how much memory you have in your computer.

  1. Choose a swap area of 50000 MB for the operating system, A Logical partition whose location is "Beginning or this space", Used as "Ext4 journaling file system", Mount point "/".
  2. Choose a swap area of whatever is left minux twice the size of your memory for the operating system, A Logical partition whose location is "Beginning or this space", Used as "Ext4 journaling file system", Mount point "/home".
  3. Choose a swap aread of twice your memory area for swap space, A Logical partition whose location is "Beginning or this space", Used as "swap space", mount point will be grayed out.
  4. press "Install Now"
  5. Choose your time zone and keyboard layout.
  6. Enter your name, a name for your computer, a user name, and a password. (You will probably also want to require login.)
  7. Ubuntu should load. With luck, you will be able to log into it when you restart.
