In this lesson I will go through the development of the first program in the W3Schools JavaScript lessons.

First, typing in the following code. This simple HTML should take about five minutes. You may write less, but do not write more. Save the code in ~/src/wp/html/lightbulb.html

<html> <head> <meta charset=”utf-8”> <title>Lightbulb example</title> </head> <body> <h1>JavaScript Changes Images</h1> </body> </html>

We will test the code by starting the server and looking at the code in the browser. If the server is not running, start it. Then point the browser to localhost:30000/lightbulb.html.. Do you see the title and the header? If not, fix the code before moving on.

Next, add a little bit more code.

<html> <head> <meta charset=”utf-8”> <title>Lightbulb example</title> </head> <body> <h1>JavaScript Changes Images</h1> <img src="img/pic_bulboff.gif"> <p> Click the light bulb to turn on/off the light. </p> </body> </html>

Refresh the your broswer window to make sure it looks right. If it does, add a little more. Do you see the picture of the lightbulb?
