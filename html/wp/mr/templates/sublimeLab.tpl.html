Installing Sublime Text

Start up a terminal window and enter the following commands into the window. This will install Sublime Text on Ubuntu Linux.

  1. sudo add-apt-repository ppa:webupd8team/sublime-text-3
  2. sudo apt-get update
  3. sudo apt-get install sublime-text-installer

The first of these lines adds an sublime-text-3 to the objects that apt-get knows about. The "ppa" at the beginning of the line stands for Personal Package Archive, which means that it was set up by an individual. Sublime text is not open source software and the author asks for 70 $US to use the software. I recommend that you move to Eclipse as soon as you have the time to learn it. Eclipse is open source. The second tells apt-get to get the information to install it and the third tells apt-get to go ahead and install the program.
